* 깃헙은 시간여행이 가능한 구글 드라이브
* 레파지토리는 폴더
* 버전은 저장 시점 또는 시간여행 시점
* 기본 명령어 4개
    * git pull: 깃헙에서 최신 코드 내려받기(구글드라이브 -> 내 컴퓨터)
    * git add: 내 컴퓨터에서 작업한 내용을 깃으로 관리해줘!(내 컴퓨터 -> 관리 영역)
    * git commit: 관리 영역에서 작업한 내용을 버전으로 만들어줘!(관리 영역 -> 버전)
    * git push: 내 컴퓨터에서 깃헙으로 코드 올리기(내 컴퓨터 -> 구글드라이브)
* 우리 수업에서 깃헙은 2일차에 초급 강의를 하고 중급, 고급강의는 3개월차에 있습니다. 왜냐하면 '익숙해질 시간'을 드리는 것입니다.
* https://github.com/paullabkorea/test -> https://github.dev/paullabkorea/test
* 자동으로 레포지토리에서 `.`(닷)을 입력하면 웹에서 편집 가능
