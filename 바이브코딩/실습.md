# 실습하기 전
- 개발하고 싶은 서비스
- 백엔드가 필요한지 체크할 필요가 있습니다.
- 백엔드가 필요하다면 어떤 서비스가 구체적으로 필요한지
- 백엔드 없이 기록만 하는 용도라면 노션이나 구글 시트로도 충분합니다.

# 0단계: 코드 없이 개발하기(아임웹, 노션 등)
- 난이도: 1
- 설명: 코드 없이 개발하는 방법입니다.

# 1단계: 백엔드 없이 프론트엔드 만으로 개발하기
- 난이도: 1
- 설명: 정적 웹 페이지를 HTML/CSS/JS로 구현합니다.
- 배포: GitHub Pages, Netlify, Vercel 등을 활용해 배포합니다.

# 2단계: 백엔드 없이 프론트엔드 + 솔루션으로 개발하기
- 난이도: 2

# 3단계: 백엔드 없이 프론트엔드 + 구글 시트
- 난이도: 5

# 4단계: 백엔드 + 프론트엔드
- 난이도: 100
