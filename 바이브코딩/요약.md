Q: 이미지 도안을 올렸습니다. 그런데 이미지 도안을 Claude가 잘 해석하지 못하고, 머메이드로도 정확히 변환해주지 못했습니다. 도움을 주실 수 있을까요?

A: ChatGPT, Sonnet(LLM) 등이 이미지 읽을 수 있나요? 생성형 AI가 PDF 읽을 수 있나요? 생성형 AI가 웹에서 데이터 가져올 수 있나요? GPT는 텍스트를 생성하는 모델입니다. 텍스트 생성 외에 다른 기능이 없습니다!

그러면 어떻게 해서 이미지를 읽었을까요?

유저 - 프롬프트 - LLM이 스스로 할 수 있는 업무인지를 판단 - 가상환경 - 여러 도구들이 있습니다.

예를 들어, '감귤 판매량 셈플 데이터를 바그래프로 샘플로 그려줘.' ChatGPT에게 물어보면 바 그래프를 그립니다. 그런데 이 그래프 ChatGPT 모델이 그릴 수 있나요? 아닙니다. ChatGPT는 텍스트 생성 모델입니다. ChatGPT가 그래프를 그릴 수 있는 것이 아니라, ChatGPT가 가상환경에 접근해서 거기에 있는 도구들에 요청을 해서 받은 데이터를 표시했을 뿐입니다.

- 가지고 있는 도구들
    - GPT는 Python 실행 환경(크롤링, 데이터분석, 압축 풀기 등을 진행합니다.), Claude는 JavaScript 실행환경
    - 엑셀이나 파워포인트와 같은 오피스 도구들
    - 압축은 GPT만 되고 Claude는 안됩니다.
    - PDF는 인식이 잘 안되는 경우도 많으니 만약 PDF에서 읽는데 제대로 못읽었다면 텍스트로 긁어서 올려주세요.

```
나는 프로젝트를 만들 때 '요구사항.md(앞 시간에 작성한 것)' 파일처럼 작성을 해야 프로덕트의 퀄리티가 향상된다는 것을 알게 되었어. 그래서 모든 단계에서 이러한 프롬프트 템플릿을 만들어 놓고 반복해서 사용하고 싶어. 조금 규모가 있는 프로젝트라고 생각하고 이러한 프롬프트 템플릿 라이브러리를 만들어줘.

각각의 단계는 폴더로 구분하고, 각 폴더 안에는 해당 단계에서 사용할 수 있는 프롬프트 템플릿 파일들을 포함시켜줘. 예를 들어, '아이디어 구상', '요구사항 정의', '설계', '개발', '테스트', '배포' 같은 단계들이 있을 수 있어. 각 단계별로 최소 3개의 프롬프트 템플릿 파일을 만들어줘.
```
